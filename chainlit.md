# Aria ✨

Your local AI sidekick for code, research, and getting real work done. 🛠️

Ask for quick answers, deep research, file edits, code help, web lookups, or bigger tasks that need a plan. 🚀

### Try asking 💡

```text
Search the web for the latest Python 3.13 features
Write a script that batch-renames images by date
Research the best open-source LLMs for code generation
Plan a migration from SQLite to Postgres and create the steps
Look up AAPL stock price and summarize recent news
Search IMDB for top-rated sci-fi movies from 2025
Scrape the content of a webpage and summarize it
Execute this Python code and show me the output
```

### Powered by vLLM 🤖

Aria runs models locally on your GPU using [vLLM](https://github.com/vllm-project/vllm) for fast, private inference — no data leaves your machine.

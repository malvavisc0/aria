# Aria

Your local AI sidekick for code, research, and getting real work done.

Ask for quick answers, deep research, file edits, web lookups, or bigger tasks that need a plan. Requests are automatically enhanced before execution.

### Capabilities

| | Capability | Examples |
|---|---|---|
| 🔍 | **Web & Research** | Search the web, browse pages, download files, look up YouTube transcripts |
| 💹 | **Finance & Data** | Stock prices, crypto, company info, market news |
| 🧠 | **Memory** | Store and recall facts across sessions |
| 📁 | **Files & Code** | Read, write, edit files; run Python; execute shell commands |
| 🎯 | **Delegation** | Complex tasks get background workers with progress tracking |
| 🔒 | **Private** | Runs locally on your GPU — no data leaves your machine |

### Try asking

```text
Search the web for the latest Python 3.13 features
Write a script that batch-renames images by date
Look up AAPL stock price and summarize recent news
Read this PDF and extract the key findings
Find all TODO comments in my project and create a summary
Remember that my preferred database is PostgreSQL
```

### Powered by vLLM

Aria runs models locally on your GPU using [vLLM](https://github.com/vllm-project/vllm) for fast, private inference.
